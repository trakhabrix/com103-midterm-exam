# Sports Tracker (Python)

## Overview

A simple Python program that records 4 sports games, calculates total points, and determines the section’s standing.

## Sports List

1. Basketball (Team)
2. Volleyball (Team)
3. Badminton (Individual)
4. Chess (Individual)
5. Table Tennis (Individual)

## How It Works

* Input section name and coordinator
* Enter 4 game entries (or `0` to skip)
* For each valid game:

  * Enter rival section
  * Enter result: `W` (3 pts) or `L` (0 pts)

## Standing

* **9+ points** → Gold Contender
* **6–8 points** → Silver Push
* **Below 6** → Keep Fighting!!
